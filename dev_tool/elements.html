What does the Element panel do?

The Elements panel provides an interface to inspect and manipulate the DOM and the CSS of your website. In the main panel it shows the DOM layout of your HTML. This allows you to see the structure of your website and make sure that all of the elements are where they are supposed to be. You are able to visually see the parent and child relationships

You can see in this example that the body element is the descendant selector and the a-tag is the child selector.
